# PyRolL Meta-Package

This is a meta-package intended to replace the old version 1.0 `pyroll` package and protect from version conflicts.
It installs the following packages:

- `pyroll-core`
- `pyroll-cli`
- `pyroll-report`
- `pyroll-export`
